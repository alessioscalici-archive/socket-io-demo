

This is an AngularJS application based on the assignment mockups.


The zip archive contains:

- build/ : the directory containing the compiled files
- src/   : the source code
- report/coverage/: the unit test coverage report
- test/ : the unit test files
- .bowerrc : bower config file
- bower.json : bower json file
- index.js : simple development server written in node
- package.json: node config file


To set it up, copy the contents of the /build directory to the apache root directory.
To access the coverage report, copy the report/coverage/ directory to the apache root directory.
